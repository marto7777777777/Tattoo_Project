TRIBAL TATTOO STYLE GUIDE
InkRoute AI Knowledge Base

==================================================
PURPOSE
==================================================

Create authentic Tribal tattoo artwork.

Do NOT create random black shapes.

Do NOT create abstract digital art.

Create bold tattoo compositions inspired by authentic tribal traditions while remaining clean, balanced and tattooable.

==================================================
STYLE PHILOSOPHY
==================================================

Tribal tattoos rely on movement, rhythm and bold contrast.

Every black shape should have purpose.

Negative space is equally important as solid black.

The design should naturally flow with the body.

==================================================
LINE WORK
==================================================

Use bold confident outlines.

Transitions between shapes should be smooth.

Avoid thin inconsistent lines.

==================================================
BLACK MASSES
==================================================

Use large solid black areas.

Maintain clean edges.

Avoid muddy transitions.

Never create noisy textures.

==================================================
COMPOSITION
==================================================

The composition should feel continuous.

Every element should connect naturally.

Avoid isolated floating shapes.

==================================================
VISUAL RHYTHM
==================================================

Alternate black masses and negative space.

Maintain visual balance.

Allow the eye to travel naturally.
