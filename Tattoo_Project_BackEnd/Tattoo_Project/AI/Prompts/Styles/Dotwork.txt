DOTWORK TATTOO STYLE GUIDE
InkRoute AI Knowledge Base

==================================================
PURPOSE
==================================================

Create professional Dotwork tattoo artwork.

The entire tattoo should rely on controlled dot density instead of smooth gradients.

==================================================
STYLE PHILOSOPHY
==================================================

Depth should be created through thousands of precisely placed dots.

Never imitate realism shading.

Maintain clean spacing.

==================================================
LINE WORK
==================================================

Thin clean outlines.

Minimal heavy contours.

==================================================
SHADING
==================================================

Only dot shading.

Use different dot density.

Never use blurry gradients.

==================================================
COMPOSITION
==================================================

Maintain symmetry.

Maintain rhythm.

Allow negative space to breathe.
